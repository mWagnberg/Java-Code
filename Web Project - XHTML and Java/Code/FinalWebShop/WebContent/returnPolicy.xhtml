Return Policy

If you are not satisfied with the product that you have received, you have 14 days to return it.

Return procedure:

  1. Do not remove any tags from the product
  2. Package the product carefully
  3. Place the sticker that was in the box on the box. The information on the sticker contains address and allows you to ship the package for free
  4. Send us an email (contact@lnu.se), specifying the order number and the reason for returning the products

After the product is received at our office, it normally takes up to 21 work days for the bank transfer to be completed; after that you should have your money back.
